1、git add readme.txt  #告诉git把文件添加到待提交的列表中
2、git commit -m "create a new file readme.txt"  #告诉git把文件提交到仓库
3、git remote add origin https://github.com/pokerc/learngit.git #将本地仓库与远程仓库关联
4、git push -u origin master #第一次推送所有内容到远程仓库
5、git push origin master  #以后推送不需要再加-u了
6、git reflog #查看版本历史
7、git reset --hard (加版本id)#全体回退到某一版本
8、git checkout --hard (加版本id) 1.py #只让1.py会退到某一个版本
9、git branch #查看分支master，dev等情况
10、git clone #克隆远程仓库项目回本地
11、git pull #将远程仓库的更新取回并和本地的进行合并
