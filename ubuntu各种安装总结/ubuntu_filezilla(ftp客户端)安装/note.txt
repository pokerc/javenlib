ubuntu系统:
1. sudo apt-get install filezilla #安装client
2. sudo apt-get install filezilla-locales #安装支持中文的语言包

然后就可以使用了。
如果出现./filezilla/filezilla.xml、./filezilla/layout.xml等文件不允许些或者说不能够读，可能是权限问题，讲这些文件赋予合适的权限就行了。

配置账户：
host栏填写服务器局域网ip地址，如10.79.27.169；username和password填自己的账户名和密码；port可以不用填(或者填21)；然后就可以连接了。

windows系统:
1.网上下载filezilla软件，并安装。
